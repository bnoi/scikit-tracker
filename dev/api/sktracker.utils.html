sktracker.utils.progress module¶

sktracker.utils.progress.print_progress(progress, out=<_io.TextIOWrapper name='<stdout>' mode='w' encoding='UTF-8'>)¶

Display a progress bar filled with a variable value. print_progress also works under IPython notebook.

Se progress to -1 to remove the progress bar.

Parameters:

progress: int

Variable value use to fill the progress bar. Should be between 0 and 100.
Returns:

None. Display the progress bar.

Examples

>>> import time
>>> from sktracker.utils import print_progress

>>> n = 5
>>> for i in range(n):
>>>     time.sleep(1)

>>>     print_progress((i + 1) * 100 / n)
100% [=================================================>]
>>> print_progress(-1)
