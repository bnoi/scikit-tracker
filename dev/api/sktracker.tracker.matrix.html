class sktracker.tracker.matrix.CostMatrix(blocks)¶

Bases: builtins.object

This class represents the cost matrix which will be given to LAPJV solver. Cost matrix is built from matrices blocks.

Parameters:

blocks : 2D list of numpy.ndarray or None

Each array value is a block or None (filled with np.nan).
