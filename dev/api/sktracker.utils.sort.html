sktracker.utils.sort module¶

sktracker.utils.sort.natural_keys(text)¶

Sort list of string in a human way.

See: http://stackoverflow.com/questions/5967500/how-to-correctly-sort-a-string-with-a-number- inside

Examples

>>> alist=["something1", "something12", "something17", "something2"]
>>> alist.sort(key=natural_keys)
>>> print(alist)
['something1', 'something2', 'something12', 'something17']
