Skicit-tracker installation¶

Dependencies¶

sktracker depends on the following packages :

  • python >= 3.3 (ask if you need support for python 2.7)
  • numpy >= 1.8
  • scipy >= 0.12
  • pandas >= 0.13
  • scikit-image >= 0.9
  • scikit-learn >= 0.13

Optional

  • matplotlib >= 1.3

For developers

  • nose >= 1.3
  • sphinx >= 1.2
  • sphinx-bootstrap-theme == 0.3.8
  • numpydoc >= 0.5
  • coverage >= 3.7

From PyPI¶

easy_install -U scikit-tracker

or

pip install -U scikit-tracker

Installation Github repo¶

Obtain the source from the git repository at https://github.com/bnoi/scikit-tracker by running:

git clone https://github.com/bnoi/scikit-tracker.git

sktracker can be installed globally using:

python setup.py install
